# **Serious**

**Serious**
is a calculator with a few simple functions, such as plus, minus, times, divided by, remainder , factorial,  and so on.

# **Screenshots**

![](http://imageshack.com/a/img921/4945/3EX9NE.png "")

# **Standard Function**

- **0~9** : NUMBER
- **+** :  PLUS
- **-** :  MINUS
- **x** : MUTIPLIED BY
- **/** : DIVIDED BY
- **=** : EQUALS SIGN
- **%** : PERCENT
- **.** : DECIMAL POINT
- **(** : LEFT RARENTHESIS
- **)** : RIGHT RARENTHESIS
- **<-** : DELETE


# Additional Function

- **AVG** : AVERAGE  
- **!** : FACTORIAL  
- **P** : DISTINTION PRIME NUMBER - RETURN TRUE OR FALSE  
- **MOD** : REMAINDER
